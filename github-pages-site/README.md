# GitHub Pages Site

这是一个用于创建个人网页的GitHub Pages项目。该项目包含以下文件和目录结构：

- **assets/**: 存放网站的静态资源，如CSS和JavaScript文件。
  - **css/**: 包含样式表文件。
  - **js/**: 包含JavaScript文件。
  
- **_includes/**: 包含可重用的HTML片段，如头部和底部。
  - **header.html**: 网站的头部部分。
  - **footer.html**: 网站的底部部分。
  
- **_layouts/**: 包含布局模板文件。
  - **default.html**: 默认布局模板。
  - **post.html**: 博客文章的布局模板。
  
- **_posts/**: 存放博客文章的目录。
  - **2024-01-01-welcome.md**: 欢迎文章。
  
- **_config.yml**: 项目的配置文件，包含网站的基本设置。
  
- **index.html**: 网站的主页。
  
- **about.md**: 关于页面的内容。
  
该项目旨在提供一个简单的个人网页解决方案，用户可以根据自己的需求进行修改和扩展。
